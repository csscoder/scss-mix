    # SCSS Functions PX to EM

    I often use this function, it's very helpful for create responsive sites.

    By default, our base font size we accept as $csscoder_base_font_size = 16px;. In my work it's a font-size for body tag.

    html {font-size: 10px;}
body {font-size: 1.6rem;}

    .test {
  font-size: em(10);
  // or you can write like this 
  font-size: em(10px); 
}

    At result, EM based on parent font size.

    .test {
  font-size: 0.625em;
  font-size: 0.625em;
}

    # Our parent has font-size not 16px

    When we have different value for parent element we must set this size;

    .test{ 
  font-size: em(24px);

  &__element {
    min-width: em(140px, 24px);
  }
}

    at result

    .test {
  font-size: 1.5em;
  /* 16*1.5 = 24 */
}
.test__element {
  min-width: 5.8333em;
  /* 24*5.8333 = 139,9992 */
}

    # We have group values in one property

    .test{ 
  padding: em(24px 44px);

  &__element {
    padding: em(12px 24px 10px);
  }

  &__element2 {
    padding: em(12px 24px 10px 8px);
  }
}

    At result

    .test {
  padding: 1.5em 2.75em;
}
.test__element {
  padding: 0.75em 1.5em 0.625em;
}
.test__element2 {
  padding: 0.75em 1.5em 0.625em 0.5em;
}

    # Work with Negative values

    .test{ 
  margin: em(24px -44px);

  &__element {
    margin: em(12px 24px -10px);
  }

  &__element2 {
    margin: em(12px 24px -10px 8px);
    box-shadow: em(4px 2px 1px) rgba(#000, .5);
  }
}

    At result

    .test {
  margin: 1.5em -2.75em;
}
.test__element {
  margin: 0.75em 1.5em -0.625em;
}
.test__element2 {
  margin: 0.75em 1.5em -0.625em 0.5em;
  box-shadow: 0.25em 0.125em 0.0625em rgba(0, 0, 0, 0.5);
}
